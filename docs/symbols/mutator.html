Class Detail
mutator()
A piecewise function manager that changes inputs over time. There are some instances where it is useful to model the transformation of a value in a complex manner. The mutator component provides functionality for simple and complex easing. You can string easing functions together to achieve more complex behavior in one instance of a Mutator through the mutation interface. For more simple easing, see sandboxe.ease. All times are delt with in seconds and the Mutator always begins at time 0.
