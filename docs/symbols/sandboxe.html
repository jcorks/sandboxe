<static> {Number} sandboxe.ease(Current, Destination, Progress, Function)
A generic easing function. For more complex easing, see mutator.
Parameters:
{Number} Current
Current value
{Number} Destination
End value
{Number} Progress
(Optional) Amount of percentage progression to that value. Default is .5f
{Number} Function
(Optional) Base easing function. Default is linear. See sandboxe.mutator.mutation

{Number} New value that should replace current.
